# Mushee for Pharos (Demo)

Heavenly, calm landing + mini-app demo for:
- **Mushee ID** (identity + reputation infrastructure)
- **Mushee Launchpad** (ecosystem launchpad for RWAs, payments, DeFi, infra)

## Links
- Website: https://mushee.cloud
- X: https://x.com/m

## Run locally
```bash
npm install
npm run dev
```

## Deploy
- Works on Vercel with default settings.
- If you later add real Pharos integrations:
  - Replace demo data in `lib/projects.ts`
  - Add your wallet connect flow + on-chain calls

## Notes
- No YC / accelerator wording.
- No blue/white glassmorphism. Background uses soft aurora + floating bubbles.
